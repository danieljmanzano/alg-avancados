# Algoritmos Avançados e Aplicações - SCC0218

Este repositório contém todas as atividades semanais e trabalhos referentes à disciplina de "Algoritmos Avançados". Cada pasta deste repositório é nomeada, de alguma forma, em referência à temática do exercício, possuindo o arquivo pdf de especificação para a atividade e o(s) código(s) que a implementam. 

Pastas notáveis são ```\atv_extensao```, ```\backtracking``` e ```\cartografo```, que são os trabalhos desenvolvidos na disciplina (que possuem certa complexidade maior do que muitos dos exercícios semanais). De toda maneira, muitos destes exercícios são extremamente interessantes e de grande complexidade de resolução.
